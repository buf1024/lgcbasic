Copy right format:
=================================
[[[[[HEAD]]]]]
Copyright information will add to top of the file

[[[[[TAIL]]]]]

Copyright information will add to bottom of the file

=================================

invoke format:
=================================
CopyRight.py Copyright_File SRC_Dir File_Pattern1 File_PatternN

The File_Pattern[1..N] use the wildcard. 


For example:
CopyRight.py copy.txt "D:\\Src" *.cpp *.java
